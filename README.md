# Website Testing

## Vision

This website will act as a **meta-directory** of projects and experiments, serving as both a creative playground and a career portfolio. The goal is to present my hyperfixations, projects, media, links, and other stuff through an **interactive desktop-like interface** inspired by classic monochrome Macintosh design.

The site will aim to simulate a desktop environment in the browser, where:

- Icons represent projects, links, or folders
- Clicking icons opens draggable windows
- Windows behave somewhat like a small operating system.

---

## Planned Features

### Desktop Interface
- [ ] Desktop-style layout inspired by classic Macintosh
- [ ] Clickable file/folder icons
- [ ] Right click menu to personalise
- [ ] Icons linking to external websites
- [ ] Icons opening internal project pages
- [ ] Icons opening directories of links/media

### Window System
- [ ] Draggable windows
- [ ] Windows open when clicking icons
- [ ] Windows can overlap (stacking/z-index)
